Admin Upload Panel (Demo)

This is a front-end demo admin panel. It does not store files because there's no backend here. To make uploads work you'll need a server-side endpoint (I can help set that up).

After uploading files to your server, add their paths to the pages or use a CMS.

Quick instructions

  1. Replace assets/img/logo.png with your real logo file.
  2. Replace other placeholder images and media in assets/img and assets/media.
  3. Host the site on any static host (Netlify, GitHub Pages, Vercel) or a shared host.
